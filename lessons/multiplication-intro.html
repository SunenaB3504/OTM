Introduction to Multiplication

Multiplication as Repeated Addition

What is Multiplication?

Multiplication is a faster way to add the same number multiple times. When we multiply, we're adding a number over and over again.

Think of multiplication as a super power for addition! Instead of writing:

5 + 5 + 5 = 15

We can write it as:

3 √ó 5 = 15

This means "3 groups of 5" or "5 added 3 times".

Everyday Examples:

  • If you have 2 hands and each hand has 5 fingers, that's 2 √ó 5 = 10 fingers total.
  • If 4 friends each have 3 cookies, that's 4 √ó 3 = 12 cookies altogether.
  • If a spider has 8 legs, then 3 spiders have 3 √ó 8 = 24 legs in total.

Multiplication helps us solve problems quickly when we're dealing with equal groups of things. It's like a shortcut for adding!

When to use multiplication:

  • When you have several equal groups of items
  • When something repeats the same number of times
  • When you want to find the total quickly

Step 5: The Multiplication Symbol (√ó)

The "√ó" symbol means we're doing multiplication.

When we write 2 √ó 3 = 6, it means the same as 3 + 3 = 6.

When we write 3 √ó 4 = 12, it means the same as 4 + 4 + 4 = 12.
