Representational Learning

Using Pictures and Visual Models to Understand Math

What is Representational Learning?

Representational learning is the second step in learning math, where we use pictures and drawings to represent math problems. These visual models help us bridge the gap between using physical objects and using just numbers and symbols.

The C-R-A Approach

Concrete → Representational → Abstract

Now that we've worked with real objects (concrete), we're ready to use drawings and diagrams (representational) before moving to numbers and symbols (abstract).
